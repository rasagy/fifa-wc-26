Three chart directions

Prototypes only, for comparison. Pick one (or mix) and I'll build it into the real artifact.

A — Slope chart: expected vs. actual rank

Each group is a small panel with four teams. A line runs from each team's Elo-expected rank (left) to its actual final rank (right). Rising = overperformed, falling = underperformed, flat = held serve.

B — Radial wheel: closeness by group

All 12 groups arranged like a tournament draw. Spoke length and color = average goal margin — short and green is a nail-biter, long and red is a blowout group.

C — Match-margin heat-strip

Every group's six matches as a row of colored cells (green = tight, red = blowout), instead of a single averaged bar. Rows sorted tightest group first.
